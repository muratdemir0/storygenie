🧙‍♂️ StoryGenie Privacy Policy

Effective Date: April 17, 2026

StoryGenie is a Chrome Extension designed to help users convert Jira issues into structured user stories using AI.

1. Data Collection

StoryGenie does not collect, store, or sell personal data.

The extension only processes data that is visible on the Jira issue page currently open in the user's browser.

2. How Data Is Used

When the user clicks "Generate User Story", the extension may send selected Jira issue content (such as title and description) to a configured AI provider.

This is done solely for the purpose of generating a structured user story.

3. AI Providers

Users can configure their own AI provider, including:

  • OpenAI-compatible APIs
  • Local models via LM Studio

StoryGenie does not control or store data processed by third-party AI providers.

Users are responsible for reviewing the privacy policies of their selected provider.

4. Data Storage

StoryGenie may store the following locally in the user's browser:

  • Selected provider settings
  • Model configuration

No sensitive data is stored on external servers by StoryGenie.

5. Data Sharing

StoryGenie does not sell, share, or transfer user data to third parties.

Data is only sent to the configured AI provider upon user action.

6. Security

StoryGenie follows Chrome Extension security best practices and does not load remote executable code.

7. Changes

This Privacy Policy may be updated in the future. Users are encouraged to review it periodically.
